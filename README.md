# AI Email Classification & Routing with n8n

This project automates customer email handling using AI and n8n.

## Problem

Handling 1000+ customer emails daily manually is slow and inefficient.

## Solution

This workflow automatically reads Gmail messages, classifies them using AI, and forwards them to the correct team.

## Categories

* Fault → Support Team
* Good Review → Marketing Team
* Package Change → Call Center
* Other → Ignored / General

## Tech Stack

* n8n
* Gmail API
* OpenAI GPT-4o-mini

## Workflow Features

* Gmail Trigger
* AI Classification
* Smart Routing
* Auto Forwarding

## Files

* `ai-email-classifier-n8n.json` → Import into n8n
